# Notes

## Tree

* / = root 

 * /root = For Root Users

 * /home = For Normal Users

 * /bin  = Have Binaries Executable by All Users
   * ls, cp, mv, more...

 * /boot = For OS Boot Files

 * /dev = For Hardware Devices Attached to System

 * /etc = For OS Configuration Files

 * /lib = For Shared Libraries Which are Often Used by '/bin' Dire

 * /media = When Attach a Device Show Here

 * /mnt = Manual Mount a Device
