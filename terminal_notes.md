# Terminal Commands

---

## Navigation Commands

```bash

# Print Working Directory #
pwd

# List #
ls
    * ls -a # list all dirs&SUB even the hidden
    * ls -l # list dirs&SUB with more info like size, privileges, etc

# Change Directory #
cd ~/Desktop/directory
    * cd .. # change to the back directory
    * .  # current
    * .. # back

```

## File Manipulation

```bash

# Make Directory #
mkdir 'dire_name'
    * mkdir '.dire_name' # make a hidden one

# create a file #
touch 'file.extension'
    * touch '.file.py' # create a python hidden file

# Copy #
cp file.py 'path'
    * cp -r # to copy dire with sub data
```
